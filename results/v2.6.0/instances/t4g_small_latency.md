# t4g.small

**t4g.small** - 2 vCPU, 2 GiB RAM (ARM-based AWS Graviton2)

## Test Configuration

- Duration: 20 seconds (`-z 20`)
- Producers: 1 (`-x 1`)
- Consumers: 1 (`-y 1`)
- Message sizes: 16,64,256,512,1024 bytes
- Rate limits: 10,100,1000,10000,50000,100000,200000,500000 msgs/s
- Queue: perf-test
- Latency measurement: Enabled (`--measure-latency`)

## Units

- Rate limits: (msgs/s)
- Latency [min, median, P75, P95, P99]: (ms)
- Publish/consume bandwidth: (MiB/s)
- Avg. publish/consume rates: (msgs/s)
